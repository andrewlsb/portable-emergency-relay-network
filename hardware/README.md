# Hardware Documentation

This folder contains wiring diagrams, connection tables, and hardware notes for each project milestone.

## Current Hardware

- Arduino UNO R4 WiFi
- STM32 Nucleo Development Board
- NEO-6M GPS Module
- SSD1306 OLED Display
- RYLR998 LoRa Modules (3)
- Breadboard
- Jumper Wires
- USB Cables

## Planned System Architecture

### Node A - Field Unit

- STM32 Nucleo Development Board
- NEO-6M GPS Module
- SSD1306 OLED Display
- RYLR998 LoRa Module
- Battery Pack

### Node B - Relay Node

- Arduino UNO R4 WiFi
- RYLR998 LoRa Module
- Battery Pack

### Node C - Base Station

- PC
- RYLR998 LoRa Module
- USB-to-UART Adapter
