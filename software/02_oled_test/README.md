# OLED Test

## Objective

Verify SSD1306 OLED display communication using I2C.

## Hardware

- Arduino UNO R4 WiFi
- SSD1306 OLED Display

## Wiring

OLED GND -> Arduino GND
OLED VCC -> Arduino 5V
OLED SDA -> Arduino SDA
OLED SCL -> Arduino SCL

## Result

Successfully displayed "HELLO" on the OLED display.
